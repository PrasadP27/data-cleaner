# Data Cleaner

Cleans the noisy data

## Installation

you will require python and nodejs

- first clone my project
- run `cd backend`
- in backend generate an virtual environment for the fast API `python -m venv myenv`
- run the virtual environment by `./myenv/Scripts/Activate.ps1` running in the same terminal
- in the myenv run this command to install dependecies `pip install -r requirements.txt`
- to start the backend server run `uvicorn main:app --reload`
- backend will start running at `http://localhost:8000/`

NOW GO TO FRONTEND

- in new terminal run `cd /frontend`
- run `npm install`
- once all the dependecies installed then run `npm run dev`
- your frontend will run at `http://localhost:3000/`

paste the noisy data and then see the magic
